# Waste-recognition
Repository for training and testing different segmentation models on multispectral waste dataset.

## Models

- UNet
- UNet++
- DeepLabV3
- DeepLabV3+
- ACNet
- MSNet

## Data structure

```md
root_dir/
└── data_folder/
    └── test/
        ├── band1
        ├── band2
        ├── ...
        └── band n
```
